Usage

Chips render each bookmaker in its own brand skin so price cards feel native, not normalized. Fortuna = yellow+black, Niké = black+orange, DOXXbet = near-black+red, Tipsport = blue+orange, Tipos = solid red, Polymarket = near-black+blue. Use these as chip backgrounds only — never as full-section fills.
